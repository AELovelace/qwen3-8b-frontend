# Rollback Functions

The following functions are used to start, join or leave a multiplayer game:

- [rollback\_join\_game](rollback_join_game.md)
- [rollback\_create\_game](rollback_create_game.md)
- [rollback\_start\_game](rollback_start_game.md)
- [rollback\_leave\_game](rollback_leave_game.md)

 

The following functions are used to define and get information for your game:

- *Set*
	- [rollback\_define\_player](rollback_define_player.md)
	- [rollback\_define\_input](rollback_define_input.md)
	- [rollback\_define\_mock\_input](rollback_define_mock_input.md)
	- [rollback\_define\_input\_frame\_delay](rollback_define_input_frame_delay.md)
	- [rollback\_use\_manual\_start](rollback_use_manual_start.md)
	- [rollback\_use\_player\_prefs](rollback_use_player_prefs.md)
	- [rollback\_set\_player\_prefs](rollback_set_player_prefs.md)
- *Get*
	- [rollback\_get\_input](rollback_get_input.md)
	- [rollback\_get\_info](rollback_get_info.md)
	- [rollback\_get\_player\_prefs](rollback_get_player_prefs.md)

 

The following functions are provided for use during an active game:

- [rollback\_chat](rollback_chat.md)
- [rollback\_sync\_on\_frame](rollback_sync_on_frame.md)

 

The following functions are used for testing, or to enable/disable features related to testing:

- [rollback\_use\_random\_input](rollback_use_random_input.md)
- [rollback\_display\_events](rollback_display_events.md)
