# player\_avatar\_url

This variable stores the URL to the player's avatar, which can be loaded using [sprite\_add()](../../Asset_Management/Sprites/Sprite_Manipulation/sprite_add.md).

You can use [player\_avatar\_sprite](player_avatar_s.md) which already has the sprite loaded.

This variable only exists in player instances created by [rollback\_define\_player()](../Rollback_Functions/rollback_define_player.md), and is different for each player.

[Guest players](player_type.md) will display a placeholder sprite.

 

#### Syntax:

player\_avatar\_url

[String](../../../../../GameMaker_Language/GML_Overview/Data_Types.md)
