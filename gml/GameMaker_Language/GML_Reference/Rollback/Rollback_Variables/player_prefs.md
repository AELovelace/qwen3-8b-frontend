# title

This variable stores the preferences set for the current player. It's assigned to player instances created using [rollback\_define\_player()](../Rollback_Functions/rollback_define_player.md).

Please read: [Player Preferences](../Rollback_Preferences.md)

 

#### Syntax:

player\_prefs;

 

#### Returns:

[Any](../../../../../GameMaker_Language/GML_Overview/Data_Types.md#variable)

 

#### Example:

image\_index \= player\_prefs.character;

The above code reads a preference and sets an instance variable in the player.
