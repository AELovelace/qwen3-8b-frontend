# vector\_sprite\_cache\_limit

This function sets the maximum size of the cache for vector sprites. The default size is 32MiB.

 

#### Syntax:

vector\_sprite\_cache\_limit(limit)

| Argument | Type | Description |
| --- | --- | --- |
| limit | [Real](../../../../GML_Overview/Data_Types.md) | The cache size limit |

 

#### Returns:

N/A

 

#### Example:

vector\_sprite\_cache\_limit(16777216\);

The code above sets the cache size for vector sprites to 16MiB.
