# vector\_sprite\_cache\_get\_limit

This function returns the current maximum size in bytes of the vector sprite cache.

 

#### Syntax:

vector\_sprite\_cache\_get\_limit()

 

#### Returns:

[Real](../../../../GML_Overview/Data_Types.md)

 

#### Example:

var \_limit \= vector\_sprite\_cache\_get\_limit();

The code above gets the current cache limit for vector sprites and stores it in a temporary variable.
