# vector\_sprite\_cache\_get\_prune\_fraction

This function returns the fraction of the vector sprite cache that is attempted to be cleared each frame.

 

#### Syntax:

vector\_sprite\_cache\_get\_prune\_fraction()

 

#### Returns:

[Real](../../../../GML_Overview/Data_Types.md)

 

#### Example:

var \_prune\_fraction \= vector\_sprite\_cache\_get\_prune\_fraction();

The code above retrieves the fraction of the vector sprite cache to be pruned and stores it in a variable.
