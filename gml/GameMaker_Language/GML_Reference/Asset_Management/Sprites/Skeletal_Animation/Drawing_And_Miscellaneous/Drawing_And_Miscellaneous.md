# Drawing And Miscellaneous

The following functions are additional tools for getting various different sets of data about a skeleton animation sprite:

- [skeleton\_get\_minmax](skeleton_get_minmax.md)
- [skeleton\_get\_num\_bounds](skeleton_get_num_bounds.md)
- [skeleton\_get\_bounds](skeleton_get_bounds.md)

 

You can also use various draw functions designed specifically for these types of sprite (apart from the regular [draw\_sprite](../../../../Drawing/Sprites_And_Tiles/draw_sprite.md) functions). These functions give you more control over what is being drawn and permit you to change animation specific features, such as timing and skins:

- [skeleton\_collision\_draw\_set](skeleton_collision_draw_set.md)
- [draw\_skeleton](draw_skeleton.md)
- [draw\_skeleton\_instance](draw_skeleton_instance.md)
- [draw\_skeleton\_collision](draw_skeleton_collision.md)
- [draw\_skeleton\_time](draw_skeleton_time.md)
- [draw\_enable\_skeleton\_blendmodes](draw_enable_skeleton_blendmodes.md)
- [draw\_get\_enable\_skeleton\_blendmodes](draw_get_enable_skeleton_blendmodes.md)
- [draw\_enable\_skeleton\_blend\_override](draw_enable_skeleton_blend_override.md)
- [draw\_get\_enable\_skeleton\_blend\_override](draw_get_enable_skeleton_blend_override.md)
