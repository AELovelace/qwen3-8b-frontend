# Animation

When working with skeletal animation sprites created using Spine, you can have multiple **animations** per sprite, and as such you need to be able to switch between them at appropriate times in your game. To help you do that, GameMaker provides the following functions:

- [skeleton\_animation\_get](skeleton_animation_get.md)
- [skeleton\_animation\_set](skeleton_animation_set.md)
- [skeleton\_animation\_get\_ext](skeleton_animation_get_ext.md)
- [skeleton\_animation\_set\_ext](skeleton_animation_set_ext.md)
- [skeleton\_animation\_get\_duration](skeleton_animation_get_duration.md)
- [skeleton\_animation\_mix](skeleton_animation_mix.md)
- [skeleton\_animation\_list](skeleton_animation_list.md)
- [skeleton\_animation\_clear](skeleton_animation_clear.md)
- [skeleton\_animation\_get\_frames](skeleton_animation_get_frames.md)
- [skeleton\_animation\_get\_frame](skeleton_animation_get_frame.md)
- [skeleton\_animation\_set\_frame](skeleton_animation_set_frame.md)
- [skeleton\_animation\_get\_position](skeleton_animation_get_position.md)
- [skeleton\_animation\_set\_position](skeleton_animation_set_position.md)
- [skeleton\_animation\_get\_event\_frames](skeleton_animation_get_event_frames.md)
- [skeleton\_animation\_is\_looping](skeleton_animation_is_looping.md)
- [skeleton\_animation\_is\_finished](skeleton_animation_is_finished.md)
