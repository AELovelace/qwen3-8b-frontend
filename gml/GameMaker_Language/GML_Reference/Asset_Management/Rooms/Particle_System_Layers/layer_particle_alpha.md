# layer\_particle\_alpha

This function sets the alpha value of the given particle system element.

 

#### Syntax:

layer\_particle\_alpha(particle\_element\_id, alpha)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| alpha | [Real](../../../../GML_Overview/Data_Types.md) | The alpha value to use, ranging from 0 (transparent) to 1 (opaque) |

 

#### Returns:

N/A
