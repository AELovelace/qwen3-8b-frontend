# layer\_particle\_blend

This function sets the blend colour of the given particle system element.

 

#### Syntax:

layer\_particle\_blend(particle\_element\_id, colour)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| colour | [Colour](../../../Drawing/Colour_And_Alpha/Colour_And_Alpha.md) | The blend colour to use |

 

#### Returns:

N/A
