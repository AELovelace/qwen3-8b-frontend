# layer\_particle\_xscale

This function sets the scale on the x axis of the given particle system element.

 

#### Syntax:

layer\_particle\_xscale(particle\_element\_id, xscale)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| xscale | [Real](../../../../GML_Overview/Data_Types.md) | The scale on the x axis of the particle system element |

 

#### Returns:

N/A
