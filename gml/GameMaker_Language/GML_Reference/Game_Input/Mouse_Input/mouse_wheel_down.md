# mouse\_wheel\_down

This function returns true if the mouse wheel is being scrolled downwards and false otherwise.

 

#### Syntax:

mouse\_wheel\_down()

 

#### Returns:

[Boolean](../../../GML_Overview/Data_Types.md)

 

#### Example:

if (mouse\_wheel\_down())  

 {  

     y \+\= 10;  

 }

This moves the current instance down in the room if the mouse wheel is scrolled downwards.
