# arcsin

This function returns the inverse sine (in radians) of x, in that if sin(x)\=n, arcsin(n)\=x, and the resulting number will be between \-pi/2 and pi/2\.

  This will only accept a value between \-1 and 1 (anything else will throw an error).

  This function returns a value in radians, not degrees.

 

#### Syntax:

arcsin(x)

| Argument | Type | Description |
| --- | --- | --- |
| x | [Real](../../../GML_Overview/Data_Types.md) | The value to return the inverse sine of |

 

#### Returns:

[Real](../../../GML_Overview/Data_Types.md)

 

#### Example:

val \= arcsin(0\);

This will set val to 0\.
