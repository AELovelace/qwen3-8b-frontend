# call\_cancel

This function cancels a callback function that was started earlier by [call\_later](call_later.md).

 

#### Syntax:

call\_cancel(handle)

| Argument | Type | Description |
| --- | --- | --- |
| handle | [Time Source](time_source_create.md) | The handle to a Time Source returned by [call\_later](call_later.md). |

 

#### Returns:

N/A
