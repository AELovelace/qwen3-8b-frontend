# title

This action will check to see if the current room is the first room as listed in the [Room Manager](../../../Settings/The_Room_Manager.md). The action will return true if the current room is the first in the Room Manager and false if it is not. You can flag the *not* checkbox to turn this into "If Room Is NOT first".

Note that to add actions into the "if" block, they should be dropped to the side of the action, as shown in the image below:

These actions will now be run if the "if" evaluates to true, while any actions
 dropped elsewhere will be performed after the "if" block.
