# Data Types Action Library

GameMaker has a number of different data types, like integers, strings, pointers, etc... and some of those can be converted to others. This section of the manual shows the different actions available for dealing with and converting different data types.

  you can find out more about the different data types [here](../../../GameMaker_Language/GML_Overview/Data_Types.md).

The **Types** library has the following actions:

|  | [Decimal To Integer](Decimal_To_Integer.md) |
| --- | --- |
|  | [String To Number](String_To_Number.md) |
|  | [Number To String](Number_To_String.md) |
