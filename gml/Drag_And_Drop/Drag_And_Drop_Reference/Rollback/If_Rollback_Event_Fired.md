# title

This action is used in the [Rollback Event](../../../GameMaker_Language/GML_Reference/Rollback/Rollback_Events.md#h1) event, to check which kind of event was fired.

It lets you attach actions on the right, which run only when the given event was fired.

 

#### Action Syntax:

#### Arguments:

| Argument | Description |
| --- | --- |
| Event | The Rollback event to check |
| Not | Enable this to check if the given event was not fired (meaning the condition will be true for all other events firing) |
