# title

This action can be used during [Sync Test](../../../GameMaker_Language/GML_Reference/Rollback/Rollback_System.md#h) to assign temporary "mock" input to the specified player. This overrides the [controls defined](Define_Input.md) for the player and any [random input](Use_Random_Input.md) from Sync Test.

 

#### Action Syntax:

#### Arguments:

| Argument | Description |
| --- | --- |
| Player ID | The player number that gets this mock input set |
| Input | The name of the input to define |
| Value | The input constant to assign (see [Custom Controls](../../../GameMaker_Language/GML_Reference/Rollback/Defining_Inputs.md#h)) |

 

#### Example:

The above action defines the mock inputs fire, left, right, up and down for player 1\.
