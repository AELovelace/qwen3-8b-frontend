# title

This function returns a struct containing information about a player. This should only be used when managing player instances manually. If you're using [Define Player](Define_Player.md), then each player instance will automatically receive this information as [variables](../../../GameMaker_Language/GML_Reference/Rollback/Rollback_Variables/Rollback_Variables.md).

For information on the contents of the returned struct, see: [rollback\_get\_info](../../../GameMaker_Language/GML_Reference/Rollback/Rollback_Functions/rollback_get_info.md)

 

#### Action Syntax:

#### Arguments:

| Argument | Description |
| --- | --- |
| Player ID | The ID of the player to get info for |
| Target | The variable where the returned struct should be stored |
