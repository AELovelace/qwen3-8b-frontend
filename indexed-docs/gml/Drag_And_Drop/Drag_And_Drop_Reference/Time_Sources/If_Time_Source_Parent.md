# title

This action allows you to control other actions based on the parent of the given Time Source.

It checks if the Time Source given in "Parent" is (or isn't) the parent of the Time Source given in "Time Source".

 

#### Action Syntax:

#### Arguments:

| Argument | Description |
| --- | --- |
| Time Source | The [Time Source ID](../../../../GameMaker_Language/GML_Reference/Time_Sources/time_source_create.md) to check the parent of |
| Not | Enable this to invert the condition |
| Parent | The [Time Source ID](../../../../GameMaker_Language/GML_Reference/Time_Sources/time_source_create.md) to compare against the parent of the previously given Time Source |
