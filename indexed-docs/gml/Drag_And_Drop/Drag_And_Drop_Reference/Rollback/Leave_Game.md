# title

This action leaves the game that is currently in progress.

See [Leaving a Room](../../../GameMaker_Language/GML_Reference/Rollback/Creating_Multiplayer.md#h) for more information.

When a player leaves a game that you are present in, the **Rollback Event** event is triggered. See [Rollback Events](../../../GameMaker_Language/GML_Reference/Rollback/Rollback_Events.md) for more information.
