# title

This action checks if the given input is held for the current player.

It lets you attach actions on the right, which run only when the given input is found.

 

#### Action Syntax:

#### Arguments:

| Argument | Description |
| --- | --- |
| Input | The name of the input to check |
| Not | Enable this to check if the input is *not* held |

 

#### Example:

The above actions move the player left or right depending on the input.
