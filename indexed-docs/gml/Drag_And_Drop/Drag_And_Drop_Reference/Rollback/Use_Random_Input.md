# title

When you play a game using [Sync Test](../../../GameMaker_Language/GML_Reference/Rollback/Rollback_System.md#h), all "remote" players receive random values for their [defined inputs](../../../GameMaker_Language/GML_Reference/Rollback/Defining_Inputs.md), as a basic way of testing. You can disable or re\-enable this behaviour using this action.

 

#### Action Syntax:

#### Arguments:

| Argument | Description |
| --- | --- |
| Enabled | Enable or disable random input for remote players while testing |
