# Set Font

With this action you can set the font to use for all subsequent draw text actions. Clicking the dialogue will open the following window where you can then select the font that you wish to use:

Note that this will set the font to use for *all* subsequent draw text calls, and
 so can be called once at the beginning of the game \- in any event, not just the Draw Event \- and all draw text actions will use the set font. However if you require multiple fonts in your game, then you will need to call this action before drawing
 any text with the required font.
