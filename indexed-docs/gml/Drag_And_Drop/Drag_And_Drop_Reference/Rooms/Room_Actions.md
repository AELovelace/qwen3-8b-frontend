# Room Action Library

The **Room** library permits you to change and check game rooms, based on the order defined in the [Room Manager](../../../Settings/The_Room_Manager.md). Note that all actions which change the room will also trigger the [Other \- Room Events](../../../The_Asset_Editors/Object_Properties/Other_Events.md) .

The following actions are available:

|  | [Go To Room](Go_To_Room.md) |
| --- | --- |
|  | [Go To Next Room](Go_To_Next_Room.md) |
|  | [Go To Previous Room](Go_To_Previous_Room.md) |
|  | [Restart Room](Restart_Room.md) |
|  | [If Room Is First](If_Room_Is_First.md) |
|  | [If Room Is Last](If_Room_Is_Last.md) |
|  | [Get Current Room](Get_Current_Room.md) |
