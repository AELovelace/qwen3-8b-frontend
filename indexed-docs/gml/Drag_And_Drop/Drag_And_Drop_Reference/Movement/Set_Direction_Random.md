# Set Direction Random

This action is used to set the movement direction of the instance to one or more random directions chosen from the 8 cardinal points. You can click one or more of the 8 directional arrows within the action and the action will choose one of them at random
 to use as the direction for the instance. If you select none of the arrows then the action will have no effect on the current direction of the instance.
