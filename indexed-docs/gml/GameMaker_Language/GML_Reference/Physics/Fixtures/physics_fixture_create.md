# physics\_fixture\_create

This function creates a new physics fixture. The returning index should be stored in a variable to be used in all further functions that are used to define and use this fixture.

 
 

#### Syntax:

physics\_fixture\_create()

 

#### Returns:

[Physics Fixture ID](physics_fixture_create.md)

 

#### Example:

fix\_Ball \= physics\_fixture\_create();

The code above will create a fixture and store its index in the variable fix\_ball.
