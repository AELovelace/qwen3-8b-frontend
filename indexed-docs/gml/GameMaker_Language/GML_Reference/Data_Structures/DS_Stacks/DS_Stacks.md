# DS Stacks

A DS stack is a so called last\-in\-first\-out (LIFO) data structure, similar to a [DS queue](../DS_Queues/DS_Queues.md). You can *push* values onto a stack and then remove them again by *popping* them from the stack, where the value that was pushed on the stack most recently is the first to be popped from it again (just think of a stack of coins, where each coin that you add has to be removed again first before you can get to the rest of the coins beneath). Stacks are often used when there are interrupts to handle, or when having recursive functions, or even when constructing a rudimentary AI for your games.
