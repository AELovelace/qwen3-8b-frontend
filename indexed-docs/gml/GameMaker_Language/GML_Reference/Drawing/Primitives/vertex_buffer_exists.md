# vertex\_buffer\_exists

This function returns whether the given vertex buffer exists in memory or not.

 

#### Syntax:

vertex\_buffer\_exists(buffer)

| Argument | Type | Description |
| --- | --- | --- |
| buffer | [Vertex Buffer](vertex_create_buffer.md) | The value to check |

 

#### Returns:

[Boolean](../../../GML_Overview/Data_Types.md)

 

#### Example:

var \_exists \= vertex\_buffer\_exists(vbuff);

The code above calls vertex\_buffer\_exists to check if a variable holds a vertex buffer and assigns the result to a local variable.
