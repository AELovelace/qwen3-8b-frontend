# vertex\_format\_exists

This function returns whether the given vertex format exists in memory or not.

 

#### Syntax:

vertex\_format\_exists(format\_id)

| Argument | Type | Description |
| --- | --- | --- |
| format\_id | [Vertex Format](vertex_format_end.md) | The value to check |

 

#### Returns:

[Boolean](../../../GML_Overview/Data_Types.md)

 

#### Example:

var \_exists \= vertex\_format\_exists(format);

The code above calls vertex\_format\_exists to check if the vertex format stored in a variable exists and assigns the result to a local variable.
