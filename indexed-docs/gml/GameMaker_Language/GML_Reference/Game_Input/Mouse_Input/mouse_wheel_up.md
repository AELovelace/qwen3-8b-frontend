# mouse\_wheel\_up

This function returns true if the mouse wheel is being scrolled upwards and false otherwise.

 

#### Syntax:

mouse\_wheel\_up()

 

#### Returns:

[Boolean](../../../GML_Overview/Data_Types.md)

 

#### Example:

if (mouse\_wheel\_up())  

 {  

     y \-\= 10;  

 }

This moves the current instance up in the room if the mouse wheel is scrolled upwards.
