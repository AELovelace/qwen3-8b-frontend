# Particle System Assets

The following functions are available when working with particle system assets, which you either create using [The Particle System Editor](../../../../The_Asset_Editors/Particle_Systems.md) or at runtime using code. These functions are *only* for dealing with an asset itself, and not for working with particle system instances at runtime in your game (for that see the appropriate functions on the section [Particles](../../Drawing/Particles/Particles.md)).

The functions available for working with a [Particle System Asset](../../../../The_Asset_Editors/Particle_Systems.md) are:

- [particle\_exists](../../Drawing/Particles/particle_exists.md)
- [particle\_get\_info](../../Drawing/Particles/particle_get_info.md)
- [particle\_add](../../Drawing/Particles/particle_add.md)
- [particle\_delete](../../Drawing/Particles/particle_delete.md)
