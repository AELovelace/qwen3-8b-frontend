# vector\_sprite\_cache\_get\_used

This function returns the current amount of memory (in bytes) used by the vector sprite cache.

 

#### Syntax:

vector\_sprite\_cache\_get\_used()

 

#### Returns:

[Real](../../../../GML_Overview/Data_Types.md)

 

#### Example:

var \_used \= vector\_sprite\_cache\_get\_used();

The code above gets the number of bytes currently used by the vector sprite cache and stores this value in a temporary variable.
