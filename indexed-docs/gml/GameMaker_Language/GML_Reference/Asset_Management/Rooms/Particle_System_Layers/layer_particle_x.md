# layer\_particle\_x

This function sets the position on the x axis of the given particle system element.

  This value is the x offset of the *element*, which is different from the value you set using [part\_system\_position](../../../Drawing/Particles/Particle_Systems/part_system_position.md).

 

#### Syntax:

layer\_particle\_x(particle\_element\_id, x)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| x | [Real](../../../../GML_Overview/Data_Types.md) | The position on the x axis of the particle system element |

 

#### Returns:

N/A
