# layer\_particle\_yscale

This function sets the scale on the y axis of the given particle system element.

 

#### Syntax:

layer\_particle\_yscale(particle\_element\_id, yscale)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| yscale | [Real](../../../../GML_Overview/Data_Types.md) | The scale on the y axis of the particle system element |

 

#### Returns:

N/A
