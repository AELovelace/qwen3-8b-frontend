# layer\_particle\_y

This function sets the position on the y axis of the given particle system element.

  This value is the y offset of the *element*, which is different from the value you set using [part\_system\_position](../../../Drawing/Particles/Particle_Systems/part_system_position.md).

 

#### Syntax:

layer\_particle\_y(particle\_element\_id, y)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| y | [Real](../../../../GML_Overview/Data_Types.md) | The position on the y axis of the particle system element |

 

#### Returns:

N/A
