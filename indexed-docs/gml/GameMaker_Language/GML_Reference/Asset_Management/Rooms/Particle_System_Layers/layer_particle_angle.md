# layer\_particle\_angle

This function sets the angle of rotation of the given particle system element.

  This value is the angle of the *element*, which is different from the angle that you set using [part\_system\_angle](../../../Drawing/Particles/Particle_Systems/part_system_angle.md).

 

#### Syntax:

layer\_particle\_angle(particle\_element\_id, angle)

| Argument | Type | Description |
| --- | --- | --- |
| particle\_element\_id | [Particle System Element ID](layer_particle_get_id.md) | The unique ID value of the particle system element |
| angle | [Real](../../../../GML_Overview/Data_Types.md) | The angle of the particle system element |

 

#### Returns:

N/A
