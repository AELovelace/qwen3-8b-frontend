# Animation Curves

Animation curves can be set up in [The Asset Browser](../../../../Introduction/The_Asset_Browser.md) as assets using [The Animation Curve Editor](../../../../The_Asset_Editors/Animation_Curves.md) and they can then be used in [The Sequence Editor](../../../../The_Asset_Editors/Sequences.md) to control tracks in a sequence. However, they can also be used in your code to change values over time in your games for anything from tweening effects on buttons, to movement, to colour changing and more. You can also create your own animation curves at run time using the functions listed here, making them a very flexible tool for your games.
