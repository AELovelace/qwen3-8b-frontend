# sqr

Multiplies a number by itself and so returns the square of that number, e.g., sqr(5\) would return 25 since 5\*5\=25\.

 

#### Syntax:

sqr(val)

| Argument | Type | Description |
| --- | --- | --- |
| val | [Real](../../../GML_Overview/Data_Types.md) | The number to square |

 

#### Returns:

[Real](../../../GML_Overview/Data_Types.md)

 

#### Example:

score \+\= sqr(dmg);

The above code will add on the square of the variable "dmg" to the score.
