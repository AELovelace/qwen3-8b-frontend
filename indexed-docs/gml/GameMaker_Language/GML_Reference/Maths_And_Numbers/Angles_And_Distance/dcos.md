# dcos

This function returns the cosine of an angle in degrees. In a right\-angled triangle the cosine is defined as *cos(val) \= adjacent / hypotenuse*, where val is one of the three angles.

The function always returns a number between 1 and \-1\.

 
 

#### Syntax:

dcos(val)

| Argument | Type | Description |
| --- | --- | --- |
| val | [Real](../../../GML_Overview/Data_Types.md) | The angle (in degrees) to return the cosine of |

 

#### Returns:

[Real](../../../GML_Overview/Data_Types.md)

 

#### Example:

val \= dcos(45\);

This will set val to 0\.71\.
